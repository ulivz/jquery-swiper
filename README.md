# Features

- Responsive.
- Fast with using CSS3 (GPU acceleration).

- At desktop side, **_drag_** to move (Support **_inertial movement_**).
- Support **_Rebound_** and **_Smoothing Scroll_**.
- All view can call API to switch view.
